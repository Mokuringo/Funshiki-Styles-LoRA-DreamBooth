# Funshiki-Styles-LoRa-DreamBooth

## 介绍

**Funshiki-Styles-LoRa-DreamBooth**训练了欸嘿酱（泛式）风格的LoRA模型和DreamBooth训练的大模型。

- `dataset`：训练集
- `experiments`：示例
- `models/(lora|dreambooth)/sample`：示例使用的模型
- `models/(lora|dreambooth)/others`：训练中途保存的其他模型
